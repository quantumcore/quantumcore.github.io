# QuantumCore

> // A cyberpunk terminal-space for offensive security and deep technical research

## About

This blog explores red teaming, malware development, threat hunting, reverse engineering, and offensive security operations. Expect deep dives into custom C2 infrastructure, reflective DLL injection, phishing simulation, and kernel-level development.

## Theme
- Feel free to use it however.

## Tech Stack

- [Jekyll](https://jekyllrb.com/) — static site generator
- [GitHub Pages](https://pages.github.com/) — hosting
- Custom CSS/JS theme — CSS variables, no frameworks

## Local Development

```bash
git clone https://github.com/quantumcore/quantumcore.github.io.git
cd quantumcore.github.io
bundle install
bundle exec jekyll serve
```

Open [http://localhost:4000](http://localhost:4000) in your browser.
